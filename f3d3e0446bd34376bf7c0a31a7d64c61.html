Excalibur是一款以剑为主题的战甲。能够使用剑刃快速斩杀敌人。这款战甲是玩家开始游戏时能够选择的三款战甲之一。
被动技能
Excalibur在Tenno中以擅长于近战著称，他在使用某些近战武器（包括 显赫刀剑）时会享有+10%的攻击速度加成和+10%的伤害加成，这些武器为:
剑，如空刃。
双剑，如空刃双刀。
侍刃，如龙之侍刃。
细剑，如技巧之剑。
上述效果可以与诸如压迫点和狂暴这样的MOD效果以加法叠加。
技能
突斩
Excalibur对前方6 / 8 / 10 / 12米半径扇形范围内的敌人进行冲刺斩击，释放技能期间Excalibur无敌，斩击时使用的武器是显赫刀剑。每次攻击造成100 / 125 / 200 / 250点基础伤害。不会暴击以及触发异常状态。
突斩的攻击生效是一个小范围的圆形区域，所以可以攻击挨得很近的多个敌人，甚至可以穿透比较薄的障碍（如Grineer护盾枪兵放出的护罩）。
基础伤害由15%的 冲击、15%的 穿刺和70% 切割伤害构成。
基础伤害受技能强度和近战连击数影响。
突斩也会受当前近战武器上装备的MOD（包括赛德娜/指数武器特殊Mod、裂鎼Mod、集团Mod。要注意，如果不是手持集团Mod对应武器时使用该技能，则技能杀死的怪物不会给集团槽充能）影响。那些能增加基础伤害的MOD，如钢铁充能和压迫点都可以增加技能攻击时的伤害。而物理和元素伤害MOD则能根据当前的基础伤害来增加额外伤害。
如果使用突斩时装备的武器是显赫刀剑（即 显赫刀剑技能为开启状态），则突斩所受到Mod的影响由显赫刀剑技能决定。
突斩不会解除幽冥守护以及狩猎库伯带来的隐身
派系伤害MOD和充能对突斩没有效果。
每次攻击都能增加近战连击数。
突斩的目标是角色前方准星所指方向的一个扇形区域内的单位。Excalibur会对范围内的敌人之间以“Z”字形迅速移动并进行斩击。浮动单位，如恶徒和鱼鹰，也会被作为攻击目标。
扇形半径受技能范围影响，实际上扇形的半径就是该技能的施法范围，能被攻击的单位是以视野为准，倘若敌人前后遮挡，并超出了突斩的攻击范围（一个小的圆形区域），那么突斩将无法攻击被遮挡的单位，但是激活的剑气并不会收到影响，会沿着触发的方向，伤害直线路径上所有敌人。
Excalibur在斩击过程中是无敌的，。
但是仍然会被异常状态影响
如果扇形区域内没有敌人，那么Excalibur就会向准星所指方位进行直线突斩，不过突进过程会收到重力影响（落点会下垂）。由于这个特性，该技能可以作为所为一种移动方式来使用（比如到达高处）。
在技能发动过程中可以按下鼠标右键进入滑行状态，可以减少重力的影响。
在技能发动过程中按下近战攻击键可以让Excalibur在技能结束后直接切换为装备近战武器的状态。
当 显赫刀剑处于激活状态中时，突斩的攻击可以发出能量波，能量波伤害为显赫刀剑的剑气伤害。
可以用该技能无伤通过激光栅栏门，但是依旧会触发警报，（Grineer的激光栅栏门会大幅降低战甲护盾和消耗所有能量）。
以及受到机关的异常状态影响
可以在跳跃、滑行、前空翻、蹬墙和爬墙中使用。
致盲
Excalibur举起他的武器并向其中注入能量。武器会释放出强烈的闪光使8 / 12 / 15 / 25米范围的敌人失明7 / 10 / 12 / 15秒。
目盲持续时间受技能持续时间影响。
效果作用范围受技能范围影响。
中招的敌人一开始会处于眩晕状态。之后目盲的过程中其头上（或其他类似的身体部位）就会冒烟。
致盲技能不会影响到躲在障碍物后方的敌人，除非敌人在Excalibur的可视范围内。刚刚逃离Excalibur不久的敌人也会受此技能影响（例如，一个敌兵看到了Excalibur，于是跑开去寻找掩体，此时虽然Excalibur已经看不到这名敌人，但只要其跑开的时间不长，仍会受本技能影响）
使用近战武器攻击目盲中的敌人会有偷袭伤害加成。若攻击过程中与敌方发生身体接触则该加成会被取消。
也可以使用近战武器直接对致盲状态中的敌人执行近战处决攻击（如果单位能够处决的话）。
但是单位未死于这次近战处决攻击，致盲状态效果在持续时间内仍然有效，但是不能继续进行近战处决攻击，但是可以进行近战攻击并且有偷袭伤害加成。
目盲中的敌方近战单位会跑向发出枪声或脚步声的地方。一些单位，如禁卫军和远古单位，会自行攻击它们认为Excalibur最近出现的地方。
目盲中的远程单位会站在原地不动或跑开寻找掩体。在此过程中，他们偶尔也会向Excalibur之前所在位置投掷手雷。某些单位，如熔岩恐鸟，会向Excalibur的最近已知地点开火。在绝大多数情况下，远程单位不会在目盲过程中切换至近战状态。
摄像头也会受致盲影响而关闭激光栅栏。
可以在跳跃、滑行、前空翻、蹬墙和爬墙过程中使用。
配合猎杀型库狛或突击型库狛来快速终结目盲的敌人。
施放动作耗时0.9秒，受天赋和速度窜升影响。
致盲在不同的头目身上会有不同效果：
某些头目会丧失行动力，无法进行任何动作，只有在受到攻击时才会有所反应：
Sprag和Ven'kra Tel在目盲时是不会进行主动攻击的，只有在受到攻击的情况下她们才会展开反击。
Lech Kril中尉中尉在目盲状态下是不会使用冰浪进行攻击的（他无法看见玩家，所以也就无法进行目标定位）。在头目战一阶段中，目盲会（短暂）中止战斗进程直至效果解除。
豺狼只有在受到攻击时才会反击。不过，它会不断改变朝向，并试图让玩家角色处于其正面扇形区域内。（关于这种反应是否为头目特有AI，还是由玩家角色移动时所发出的声响造成的，还有待考证。）
Alad V在目盲时不会对玩家造成伤害，但他仍然可以发动闪光攻击来使玩家陷入目盲状态。该头目的同伴Zanuka仍然可以在目盲状态下输出伤害。它能发射小型火箭弹。虽然在目盲中它是无法进行发射的，但是如果它在目盲前就已经将火箭弹射出，那么这些火箭弹还是可以锁定跟踪玩家的。
其他头目仍会进行一些动作，但不会进行攻击：
海军陆战队中士有时会隐身，不过除此之外他不会进行其他动作。
Grustrag三霸在最初的眩晕过后寻找掩体躲起来。
Vor上尉和堕落Vor在目盲中仍然可以进行瞬移。
山猫在目盲时不会攻击玩家。不过，如果它的山猫炮塔没有被致盲，它们还是可以发动攻击的，并且山猫自身只有在所有山猫鱼鹰都被破坏的情况下才会受到伤害。
Vay Hek议员仍会会不时地在切换无敌状态，并且在受到伤害时会回避，不过他完全不会进行反击。他放出的卫星冲击无人机如果没有没被致盲，则还是会对玩家发动攻击的。
标枪
Excalibur在15 / 18 / 22 / 25米范围内的 5 / 7 / 10 / 12 个敌人身边召唤出一只标枪。随后，Excalibur将自己的武器插在地上，标枪就会向目标发射。每根标枪造成500 / 650 / 800 / 1000总伤害。
总伤害由 冲击,  穿刺和 切割三种伤害加起来得出。每种伤害数值均等。
伤害受技能强度影响,但是标枪数量却不会受这个影响。
受到本技能伤害的敌人会暂时处于眩晕中。
作用范围受技能范围影响。
尽管标枪不会穿透物体或敌人，但是它们的射击角度都是经过优化的，以保证它们可以在各种情况下（包括墙壁，掩体和盾枪兵的盾牌）命中目标。
标枪不会攻击屏障后的目标，除非在Excalibur视野范围内，或者敌人曾经看到过Excalibur。
标枪外观上和能量化的空刃相似。Excalibur Prime的标枪则是像能量化的空刃Prime。
约有1秒的施展时间。施展后的收尾动作耗时约2.5秒。
显赫刀剑
Excalibur召唤显赫刀剑代替近战武器，并切换至近战武器。
技能释放及维持：
开启时消耗25点能量。并受技能效率影响。
维持技能每秒钟消耗2.5点能量。受技能效率和技能持续时间共同影响。这个能量消耗不会通过预言Mod的组合效果转换成护盾。
Excalibur无法在本技能激活时通过 Trinity的  能量吸取、裂隙位面、团队能量回复、能量虹吸、Zenurik专精的活力冲刺来回复能量。
玩家仍然能通过能量球、 充沛赋能、狂暴化或者猎人肾上腺素 来恢复能量。
开启此技能时，你可以自由切换武器，但只要技能仍处于开启状态就会持续消耗能量。
技能没有持续时间，会持续到你再次按下技能键（默认为4）手动关闭或能量值不足强制关闭。
可以在不中断移动或冲刺的情况下使用本技能。
显赫刀剑有自己专属的面板和MOD配置和不可更换的架势。此外，显赫刀剑的基础伤害还受到
技能强度
影响。
显赫刀剑的基础数据：
普通攻击在2米的范围内造成100 / 125 / 200 / 250基础伤害。
蹬墙攻击造成400 / 500 / 800 / 1000基础伤害。
滑砍会消耗25点能量造成214 / 267.5 / 428 / 535基础伤害
飞跃和砸地攻击造成200 / 250 / 400 / 500 基础伤害，另外砸地攻击还能在5米范围内造成额外的50点 冲击伤害。砸地攻击的范围伤害会随距离衰减，并且不会造成暴击，其伤害也不受近战连击倍率的影响。砸地攻击可以使范围内的敌人被击倒。
近战攻击有2.0x的暴击伤害和15%的暴击几率，以及15%的触发几率。
近战攻击基础伤害由15%的 冲击伤害，15%的 穿刺伤害和70%的 切割伤害组成。
显赫刀剑架势的一些数据：
地面处决攻击能造成400%的普通攻击总伤害。有操作提示的近战处决攻击能造成6400%的普通攻击总伤害，并且这个伤害会转化为处决伤害。
作为一把单手剑，显赫刀剑会受到Excalibur被动技能——剑术宗师的加成。
技能强度直接影响基础伤害倍率，因此和增加伤害的MOD（如压迫点）以乘法关系计算。
作为近战武器，显赫刀剑能够触发战甲赋能。由于你原本的近战武器被替换掉了，所以你原本武器上的神威赋能（如果你装备了）无法生效。
显赫刀剑具有一些一般武器没有的效果：
滑砍时会消耗25能量并且致盲5米半径内敌人6秒。
致盲持续时间受技能持续时间,致盲范围技能范围影响。
相较于 致盲，范围和持续时间只有一半。且这个效果造成的致盲无法使敌人进入能被处决的状态。
普通、滑行和飞跃攻击都会向准星方位射出一股能量波。能量波的面板与普通攻击相同。
能量波对每个敌人造成的伤害独立计算，即分别计算暴击等级和触发异常数量。能量波和攻击直接命中的伤害也是独立计算的。
能量波持续2.5秒，每秒飞行15米。会贯穿场景障碍物。也会贯穿敌人，但每贯穿一个敌人都会减少0.5秒持续时间。
能量波命中敌人不会增加连击数或刷新连击持续时间。
被能量波击中的敌人会陷入蹒跚状态。
能量波伤害随着距离在 0 到 6.5m 之间从 100% 到 85% 线性下降，在 6.5m 到 37.5m 之间从 28% 到 0% 线性下降。
能量波的范围，速度，衰减，持续不受MOD和战甲属性影响。
作为技能，显赫刀剑的所有攻击都能无视裂隙位面造成伤害。
技能联动：
突斩:
当 突斩命中敌人时本次攻击也会带上能量波。
突斩同样能造成暴击伤害或触发状态效果。
