Volt能够掌控雷电，他如闪电般迅疾，运用电能阻拦来袭的火力，亦或是释放毁灭性的电磁风暴。他也是三款初始可选的战甲之一。
被动技能
当Volt在地面上移动（包括走动，冲刺和滑行，不包括壁面蹬冲），就能为下一次武器攻击或技能积蓄 电击伤害。
每移动1米，Volt就会积攒5点 电击伤害，最高则积累到1000点 电击伤害。
使用K式悬浮板在地面上滑行时也会积蓄伤害，但只有在Volt离开滑板后才会显示增加的静电伤害。
被动积蓄的 电击伤害不和其他元素伤害组合形成复合元素伤害，而是单独计算为 电击伤害。
技能强度和武器MOD不会影响被动积蓄的伤害，暴击和狙击枪的连击计数会影响被动积蓄的伤害，多重射击中仅有一个弹头会享受到被动积蓄伤害的加成。
积蓄的伤害不会影响经由Helminth移植而来的技能和电击加速的伤害。
技能
电击
向准星处发射一发电击弹，并向附近生成电弧，连环击中目标周围7 / 10 / 12 / 15米范围内的2 / 3 / 4 / 5个敌人，对目标和电弧链接的敌人造成75 / 100 / 125 / 200  电击伤害。
能量消耗受技能效率影响。
电弧的范围受技能范围MOD影响。
伤害值受技能强度MOD的影响。
使用技能不会中断正在进行的上弹以及其他一些动作。
电击的目标点可以是场景表面或是敌人。
弹射次数不受Mod影响，若直接对准一名敌人施放，那么技能一次最多能击中3 / 4 / 5 / 6名敌人。
电弧优先链接至离技能释放位置最近的敌人。
电击只能对处于其视线范围内的敌人进行连环电击。也就是说，电弧不会对其视野中被遮挡的敌人展开攻击，比如被障碍物背后的敌人。
受到伤害的敌人会100%触发1层 电击异常状态。被技能击中的敌人会有短暂的眩晕。
被动积蓄的伤害能以加法形式加入技能伤害，例如被动积蓄1000点 电击伤害，技能面板为200点 电击伤害，那么使用技能能造成1000+200=1200点 电击伤害。
技能联动:
电力屏障:
对 电力屏障使用电击可以让屏障充电，让所有触碰屏障的敌人受到100%的 电击技能伤害，例如，对屏障使用的 电击技能伤害为260点，那么触碰屏障的敌人会受到260点 电击伤害。
电能释放:
对受到 电能释放影响的敌人释放技能，能对连锁范围的所有敌人造成一次同等伤害的电击，不会造成触发效果。

加速
Volt对一定范围内的队友充能，提升自身和所有范围内的队友10% / 15% / 25% / 50%的移动速度和攻击速度， 8% / 13% / 15% / 17% 的弹药装填速度，效果持续5 / 6 / 8 / 10秒。
速度增幅受技能强度影响。
效果持续时间受技能持续时间影响。
技能所带来的攻击速度加成会以加法形式与其他攻击速度加成合并。举例来说，狂暴与满级的加速在+30%技能强度的情况下，能提升0.3 + 0.5 × 1.3 = 95%的攻击速度。
技能所带来的移动速度加成会以乘法形式与其他移动速度加成合并。举例来说，冲刺与满级的加速在+30%技能强度的情况下，能提升(1 × 1.3) × (1 + (0.5 × 1.3)) − 1 = 114.5%的冲刺速度。
由于技能会提升冲刺速度，所以滑行和其他依赖冲刺速度的移动方式的移动距离都会提升。
技能所带来的装填速度加成会以加法形式与其他装填速度加成效果合并。举例来说，持续火力与满级的加速在+30%技能强度的情况下，会将Volt的装填速度缩短 0.17 × 1.3 + 0.48 = 70.1%。
本技能不会影响技能跑酷速度、技能施放速度、倒地起身速度和切换武器速度等。
技能的效果可作用于Volt和其他友方单位。
只有范围内的队友才能享受加成。
队友可以主动使用后翻来结束加速状态
技能没有释放动作，可以在不中断许多动作的情况下施展，包括装填。
可以重复释放来刷新技能时间
电力屏障
Volt展开一道持续存在10 / 15 / 20 / 25秒的能量屏障，屏障能够阻挡敌人的子弹，也能增加穿过屏障的子弹的伤害。
屏障的高度和宽度不受技能范围影响。
屏障持续时间受技能持续时间影响。
虽然屏障能阻挡绝大多数敌方火力和范围伤害，但是某些自带穿透的武器，如鬣猫驯兽师的伊格尼斯，就可以穿透屏障伤害到玩家。
屏障共有两种模式。
静止屏障
流动屏障
创造一面4.25米高、6米宽的屏障来抵御敌方火力。
由于电力屏障会以玩家准星位置为中心出现，玩家可以以此来控制屏障的朝向。
可以重复多次施放，但屏障最多同时存在6个。超过这个上限后，新屏障将会导致原来最早施放的屏障消失。
在手持流动屏障时，Volt无法施展技能放置静止屏障。
Volt及队友射出的子弹在经过这道屏障后会获得50%额外 电击伤害，并且其暴击伤害也会乘以200%。
这些伤害增幅不受技能强度MOD影响。
电击伤害加成与元素伤害类MOD类似。举例来说，对于一把基础伤害为100的武器，其子弹在经过一道屏障后的伤害会变为100 × (1 + 0.5) = 150。
电力屏障的 电击伤害不和其他元素伤害组合形成复合元素伤害，而是单独计算为 电击伤害。
子弹在穿过多道屏障后， 电击伤害加成就会叠加。比如，在穿过3道屏障后 电击伤害加成就会变成150%。暴击伤害倍率不会随穿过的屏障数量而增加。
伤害增幅对于武器来说会有以下作用：
所有的主武器和副武器都能得到 电击伤害加成。但投掷类型的近战武器，如战刃，是不会受电力屏障影响的。
对于那些有多种伤害方式或是多段伤害的武器，只要其伤害可以受到Mod增幅，电力屏障就能对这些伤害产生影响。
如托里德发射出的榴弹在通过屏障后，其弹头碰撞伤害和爆炸的范围伤害都会得到提升。
异融量子枪所产生的球体在通过屏障后，其接触伤害不会发生变化（这个伤害无法通过装备MOD来提升），但屏障可以提升球体的爆炸伤害。
持续射击型的武器的子弹会得到瞬间命中的效果。比如光谱切割器的激光在正常情况下是有一定的飞行时间，但在穿过屏障后，它就可以瞬间命中目标。
屏障与部份战甲技能的互动关系：
屏障的 电击伤害加成对 Mesa的 和平使者的手枪有效。
战甲技能若是能以投射物形式穿过屏障，那么也会受到伤害增幅，如 Ember的 火球、 Nova的 反物质释放
对 Gara的 琉璃碎击、 Excalibur的 显赫刀剑的剑气无效。
会挡住 Nova的 湮灭流星群
技能联动：
对屏障施展 电击会使其进入带电状态，此时穿过屏障的敌人会承受100%的 电击技能伤害（如果电击能造成125点伤害，那么屏障也会造成125点伤害）。Volt的被动能力所带来的伤害加成也会计入这个电击的伤害中。
电能释放
Volt释放一股能量影响半径12 / 15 / 18 / 20米范围的强大电能脉冲。
脉冲范围受到技能范围影响。
使用电能释放是全身性动作，有大约一秒的施放时间，会打断其它动作，可以在空中使用。在施放动作中的Volt免疫击倒和击晕。
在最初施放的3 / 3 / 4 / 4秒内无法再次使用，该冷却时间不受技能持续时间影响
重复施放技能不会刷新或覆盖已被效果影响的敌人身上的任何技能效果。
Volt释放脉冲，使得范围12 / 15 /18 / 20米内的敌人被“线圈化”，该效果会持续3 / 4 / 5 / 6秒。
被特斯拉线圈化的敌人会向周围释放电弧，对范围内敌人造成每一周期250 / 375 / 500 / 600的 电击伤害，电击伤害周期为0.5秒。被特斯拉线圈化的敌人会在放电期间里持续被击晕。
能够造成的伤害量随距离衰减，在最远处衰减为50%。
电弧伤害受到技能强度影响，电弧伤害本身不会衰减。
电弧释放范围为半径4 / 4 / 6 / 8米，受到技能范围影响。
效果持续时间受技能持续时间影响
若两个相邻的敌人都被“线圈化”，那么他们在完整的技能持续时间里会互相电击。
若是被“线圈化”的敌人电击其它敌人，则只会电击一半的持续时间。
电弧伤害的伤害周期不受技能持续时间影响，不会对敌人造成 电击触发效果。
电弧不会被环境阻挡。储存容器也会受到特斯拉线圈化效果影响，且只会被影响一次。
4.5秒之后，会以Volt当前位置为中心，将线圈化的敌人里“未放电”的敌人“激活”。受到“激活”的敌人会在效果期间里持续被击晕。
即线圈化的敌人在先前的4.5秒内电击过其他敌人，那么它不会受到“激活”。
线圈化的敌人里“未放电”的敌人在4.5秒之后会解除其线圈化效果。
4.5秒的延迟不受技能持续时间影响，如果技能持续时间被压低到不足4.5秒（即75%技能持续时间以下），则敌人会提前解除线圈化，从而不会造成“激活”伤害。
以Volt当前位置为中心，意味着在4.5秒的延迟后，玩家离敌人太远也会导致不造成“激活”效果。
被“激活”的敌人自身会受到每一周期250 / 375 / 500 / 600的 电击伤害，直到剩余的技能时间结束。“激活”伤害周期为0.5秒。
剩余持续时间会随距离衰减，最大衰减至原本剩余持续时间的20%，计算衰减的起始距离取决于技能释放时的位置，而不是Volt当前位置。
“激活”伤害受到技能强度影响。
“激活”伤害的伤害周期不受技能持续时间影响，不会对敌人造成 电击触发效果。
Volt的被动技能会增加 电能释放的伤害，但只会影响“线圈化”的电弧的周期伤害，不影响“激活”伤害。
例如，三级的 电能释放配上满级的聚精会神，在不考虑衰减的情况下，每周期伤害应为600 * 1.3 =780点。在有1000点Volt的被动技能伤害时，在不考虑衰减的情况下，电弧每周期伤害为1000 + (600 × 1.3) = 1780。
技能联动：
对被线圈化的敌人释放 电击，能对技能连锁范围的所有敌人造成一次同等伤害的电击，不会造成触发效果。
