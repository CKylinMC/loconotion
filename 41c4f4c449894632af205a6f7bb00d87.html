Mag是一款运用磁力战斗的战甲。她能够操控磁场为友军提供保护，或是粉碎敌人。她是玩家一开始就能选择的三款战甲之一。
被动技能
Mag在进行旋身飞跃的时候会将周围的可拾取物吸过来，其效果与搬运者的吸取类似。
技能
吸引
Mag制造强大的力场，将半径 6 米内的所有敌人拉扯到自己周围，并以瞄准方向90° 角度为方向拖拽 15 / 20 / 22 / 25 米内的敌人，受到影响的敌人会被击倒且造成布娃娃效果并以 ？ 的速度拉向Mag，同时对他们造成 100 / 125 / 200 / 300 点 磁力伤害。被该技能杀死的敌人有 10 %/ 15% / 20% / 25% 的额外几率掉落能量球。
伤害和额外能量球掉落几率受技能强度影响。
敌人受到的拉力大小受技能强度影响。
拉扯距离受技能范围影响。
牵引范围实际上是个Mag面前的锥形和一个围绕Mag的小圈，在身后的敌人不会被牵引除非他们处于这个小圈中。
拉扯半径和角度不受mod的影响。
技能施展耗时0.5秒，并且在连续使用时有0.25秒的冷却。受到 天赋 或 速度窜升 影响。
不会拉近一些特定敌人，如鱼鹰和头目。
牵引不会打断一些敌人已经释放到一半的技能，例如重型Grineer族的震地和爆炸奔跑者的自爆。
能力协同： 如果敌人正处于 磁吸力场的影响下，则受到的伤害会加倍。
本技能为单手动作，因此它不会中断装填、蓄力或射击这类动作。
可在半空中释放，但不能在滑索上释放。
磁吸力场
磁吸力场在轻按和长按时有不同的使用模式。
当Mag瞄准敌人时，轻按能力键(默认为2)可以在敌人周围创建一个磁场。长按时，Mag在她的双手之间创造了磁性奇点，保护她免受敌人的攻击。无论哪种形式，磁吸力场都需要消耗 50 能量，持续时间为 10 / 12 / 13 / 15 秒。
持续时间受技能持续时间影响。
能量消耗受技能效率影响。
施法范围 不 受mod影响，轻按瞄准距离无限。
1秒的施放动画会受到天赋 和 速度窜升的影响。
当磁性奇点形成后，放开按键 不会 使磁性奇点消失。

防卫（长按）
产生一个强大的磁场包围并锚定目标，半径为2 / 3 / 3.5 / 4 米 ，并以 2x / 2.5x / 3.5x / 4x 的磁力强度拖拽受影响范围内的所有敌人。在磁场作用期间，所有的投射物、子弹、以及 护盾极化产生的碎片将被拉到磁场的中心，并将有125% / 150% / 175% / 200%的伤害加成。此外，磁场对范围内所有敌人每秒造成所吸收伤害的10 / 15 / 20 / 25%。如初始目标死亡，磁场会在技能结束时因不稳定而爆炸，产生50 / 150 / 200 / 300  爆炸伤害，范围5 / 10 / 12 / 15米。此爆炸伤害会加上磁场所吸收的总伤害的125% / 150% / 175% / 200%。
伤害倍数和爆炸伤害受技能强度影响
爆炸伤害随范围递减，并且不会穿过掩体。
磁场持续时间受技能持续时间影响。
磁场范围、爆炸范围受技能范围影响
满级技能对伤害的吸收倍率固定为0.25，不受技能强度影响。
重定向的弹丸 从磁场（这个球体）的撞击点到磁场中心成一条直线，这就避开了敌人弱点。然而，你可以瞄准球体上任一垂直于弱点的地方，来攻击致命区域。点射武器不难集中弱点，只瞄准目标的弱点就可以轻松产生大量伤害。
在磁场内发射子弹也会被重定向，包括被磁化的目标发射的子弹。 除了枪支，吸引力可将： 标枪的刀刃、 棱镜的激光、弹跳弹丸-例如战刃，重定向成一个单一的目标。
不能重定向范围伤害技能（包括赋能效果）和瞬时命中类枪刃的子弹。
技能施放动作耗时1秒。
如果目标在磁吸力场出现前就已经死亡，那么磁吸力场所消耗的能量会返还给Mag。
该技能是单手动作，因此，它不会中断诸如装填，蓄力或射击。
可以在空中使用。
护盾极化
Mag创造一个初始范围为 6 米的磁能脉冲，以 6 米/秒的速度向外传播 2 / 3 / 4 / 5 秒。如果敌人拥有 护盾 对其造成250 / 300 / 350 / 400伤害。如果敌人拥有护甲，则移除其 250 / 300 / 350 / 400 点护甲，并对其造成等同于护甲移除值的 真实伤害。此外，如果友军在初始半径内或接触到能量脉冲，将会恢复 250 / 300 / 350 / 400 点 护盾。
造成的伤害、护盾回复、护盾损耗、护甲损耗以及碎片伤害倍率均受技能强度影响。
初始半径受技能范围影响。
由于磁能脉冲在初始半径之后开始传播，因此也会略微增加总半径。
脉冲传播时间受技能持续时间影响。
初始半径和传播距离总和为：总半径 = (1 + 技能范围) × 6 + (1 + 技能持续时间) × 5 × 6.
装备满级 延伸 和 持久力, 等级3护盾极化的总半径为 (1 + 0.45) × 6 + (1 + 0.3) × 30 = 47.7 米。
磁能脉冲传播速度 不受 Mod影响。
有1秒的施放时间，并有0.2秒的施放间隔，会受到天赋 和 速度窜升的影响。
对护甲类敌人造成的伤害也会削减其与技能伤害等量的护甲值。如果一名敌人同时用有护盾以及护甲的情况下，那么只有他的护盾完全耗尽后，才能对其护甲产生损耗。
恢复MAG的护盾，以及同队的Tenno、同伴、人质、防御模式下冷冻舱/神器的护盾、从恐鸟储存柜劫持而来的恐鸟的护盾。
在对于单一敌人的情况下，该技能仅会对与其护甲或护盾造成伤害，并不会直接伤害其生命值。
每个被耗损的目标都产生一个猛烈的磁能爆发，对初始半径内和被损耗目标半径5 / 6 / 7 / 8米内的所有敌人造成其护盾或护甲损耗值的1x / 1.5x / 2x / 2.5x的 磁力伤害。
该伤害倍数受能力技能强度影响，爆炸伤害随距离衰减。
该伤害可穿过障碍物，并能够击中弱点部位。
磁力 对护盾 和 原型护盾 非常有效, 但是对 合金装甲效果较差。
爆炸半径受技能范围影响。
被损耗的敌人不会被自身爆炸所伤，但会受到邻近因护甲或者护盾被损耗而产生的碎片影响，从而受到碎片的爆炸伤害。
能力协同：
如果接触脉冲的敌人是 磁吸力场的目标，则对其造成 200％ 的效果。 这 不会 影响那些被吸入磁吸力场的其他目标。
伤害增幅受技能强度影响。
每个护甲或护盾被损耗的敌人会掉落1或2片储存50点伤害的碎片，当与 磁吸力场相接触的时候会被其吸引并释放碎片储存的伤害。
碎片储存的伤害和掉落的数量不受技能强度影响。
每个碎片在场上独立存在30秒。
本技能为单手动作，因此它不会中断装填、蓄力或射击这类动作。
可以在空中施放，但是不能在滑索上施放。
粉碎
Mag磁化半径 8/10/13/18 米内敌人的骨头, 将它们举起并砸向地面造成 800 / 1000 / 1250 / 1500  磁力伤害，总共造成3次打击。 受 磁吸力场影响的敌人会承受额外800 / 1000 / 1250/ 1500 磁力伤害。
伤害受技能强度影响
伤害会被三等分并在施展动作过程中依次施加。
在技能施放中走入区域内的敌人会受到接下来尚未施加的伤害，同时也能恢复护盾。。
被举起的敌人在释放过程中无法行动，并且在受到最后一次伤害后会被击倒。
装备满级 盲怒 和 聚精会神, 等级3粉碎将会造成((基础伤害 ÷ 3) + 额外伤害 ) × (1 + 技能强度) 磁吸力场 附加伤害 = ((1500 ÷ 3) + 1500 ) × (1 + 1.29)^2 = 10,488.2 对每一个受到  磁吸力场 的目标。
磁力 对护盾 和 原型护盾 非常有效, 但是对 合金装甲效果较差。
距离受技能范围影响。
有1.5秒释放时间，受天赋和速度窜升的影响。
施放粉碎会中断任何其他动作，并强制Mag施展整个动画效果。
此外，在击中敌人的情况下，所有在技能释放半径内的除了防御目标和人质外的友方都将获得40点基础护盾，再加上范围内每段技能每命中一名敌人，额外恢复10 / 15 / 20 / 25点护盾。
产生基础护盾值不受技能强度的影响，但是范围内每段攻击攻击到敌人额外恢复的护盾数值受其影响。
可以产生过载护盾。
在技能施放中走入区域内的敌人会受到接下来尚未施加的伤害，同时也能恢复护盾。
被举起的敌人在释放过程中无法行动，并且在受到最后一次伤害后会被击倒。
可以在空中施放了。
Tenno对战时，此技能会造成150点伤害并击倒对手。
视觉效果受到Mag所选择的能量颜色的影响
Mag Prime在施放本技能时，技能的视觉效果为金色。
