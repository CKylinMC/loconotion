加入氏族必读
ShadowKylin氏族是一个Warframe(国际服)简中区的氏族，从2018年2月初建至今已有近3年的历史。
最初的SK氏族只是一个熟人圈子，几个互相认识一起玩Warframe的朋友一起建立了在warframe中的小基地，随着加入的人慢慢变多，氏族也扩展到了如今的在游戏成员数十人、历史成员数百人的规模。
人越来越多也就意味着不能再像以前那样一盘散沙，要进行一定的管理。因此，指定了一些氏族的规定，要求每个成员认真阅读并履行。
为了方便管理，我们要求要加入氏族的成员必须加入我们的QQ群。我们会依据QQ群成员情况来管理氏族内成员，如果您退出QQ群，我们将同时将您移除出氏族。
QQ群：818581951
当您加入QQ群后，请按照群规定或按照下面的格式修改你的群名片：
[段位] 游戏ID 如：[21]CKylinMC
进入群后，您可以寻找人员管理员来将您邀请到游戏内氏族。
游戏内邀请会被发送到游戏中的收件箱，需要手动确认才能进入。
收件箱位置：ESC-通讯-收件箱
如果您有任何问题，可以直接在群里提问，群成员都很乐意分享他们的见解。但是如果他们没有回复您，并不是因为您不受欢迎，而是因为这些问题可能他们也刚好不是特别了解。
我们推荐每一个新人玩家遇到问题时都优先前往维基网站或b站等查询攻略和介绍，因为那里的是最准确的。
每个人对游戏的理解、每个人对游戏了解的程度、每个人的游戏习惯都不同，或许别人告诉你的并不适合你，我们建议在遇到配卡等问题时，先参考别人的配卡，然后再进行修改。哪怕效果不如意，也不要着急否认别人，因为有可能只是你缺少了别人认为很常见的某种条件，也有可能是因为版本迭代很快，有些很可能不再适用于现版本。
我们总结了一份新人常用的工具站点列表，以及常见问题的教程列表，一般可以解决您的大部分问题：
新手资源总览

我们目前不强制捐献任何资源也不曾这么做，但这不代表您不需要捐献任何资源。
很多资源都是关卡掉落后并不立刻使用的资源。如果您看到氏族正募集这些资源，我们诚恳的希望您能尽一份薄力，一起为氏族的发展做一份贡献。
虽然哪怕您只在氏族挂名然后进行单人游玩也没有任何问题，但是显然在集体中，不做任何贡献的人一般不会受到欢迎。

进入氏族道场需要去制造厂制作道场钥匙。我们不建议新手期的玩家加速道场钥匙的制作。
氏族道场的功能很多，比如氏族研究室可以复制很多氏族限定的武器或战甲以及物品的蓝图，也可以进行免额外税的交易（标准税是必须的，并且不会进入氏族金库），还可以进行航道星舰的调控。
一般来说，您不需要熟悉氏族道场的地图和构造。(事实上哪怕是我们也背不下来这乱七八糟的构造，氏族长背锅)您只需要在道场按下ESC键，然后选择快速移动即可。

请遵守游戏规则进行游戏。
淘宝确实售卖便宜的游戏货币（白金），但是我们管这个叫黑金，因为来源不正当，而且必定会导致您的账户被永久封禁，买到账户的那一刻开始，最久不会超过两个月。
大部分情况下，白金都可以通过游戏内的方式获取，不需要额外充值。
如果需要充值加速游戏体验，请耐心等待签到赠送的“-75%”折扣后再前往官网充值。充值请务必关闭加速器等软件，此类软件会导致您的地区错误，造成跨区充值，极易被封号。
我们不是不推荐使用加速器，相反，如果想保证游戏体验我们推荐您至少根据情况买一个加速器软件使用。但是请注意，由于加速器原理限制，涉及充值等操作请务必关闭后再继续操作。
我们反对包含真实RMB交易的任何交易行为，这违反了warframe的最终用户许可协议。这个行为被称为RMT(Real Money Trading)。
理性游戏，劳逸结合，请勿上头。
紫卡不出好词条、关卡不掉需要的东西都是很正常的事情。
网络连接无响应、主机漂移、bug、吞结算也都是再常见不过的事，放平心态，大家都是这样过来的。
如果您很介意请转到warframe国服(星际战甲)。
绝对不建议一口气猛肝到底。游戏只是数据，身体是自己的。刷子游戏就是拖你的时长，请不要用命来换。
关于AFK：氏族允许一定程度的不活跃行为，也不会因为不活跃就完全把您从所有联系方式都移除。我们可能会因为您不活跃而从游戏内移除，为其他成员留出空位。具体请见我们的AFK政策。
最后，这一点并不是规则中的一项，而是在我们带了很多届新人后发现的问题，而提出的一点小小的建议。
我们希望您在游玩这个游戏的时候，有比其他游戏更充足的耐心。这是一个刷子游戏，是一个肝向游戏，每一个环节的设置都旨在延长你的游戏世界。这意味着只要你不花钱跳过，那么你就需要花费大量时间和精力在某些事上。我们见过很多玩家过于追求提升速度和游戏进度，这对新人来说是一个绝对的坏事。游戏机制很复杂，要了解的事情很多，我们建议您多沉下心来好好了解一下这个游戏，这个故事。如果您真的耐心的认真的游玩了每一个任务，您到后期就一定不会觉得好像什么都不了解。虽然游戏提示做的很糟糕，但是至少确保了所有关键的信息都至少出现过一次。关于新人应该如何开始玩warframe我们会单独写在攻略中。但是，我们还是希望，您在游玩时，多一些耐心，多一些思考，而不是无脑玩游戏。
这个游戏大部分到中后期的玩家都有几千小时的游戏时长。不是说短就不好，也不是说可以拿来炫耀，而是这个游戏真的需要这么长的时间来理解。如果您到游戏后期，时长却短于600小时，不妨回头看一看，自己是不是错过了那些最精彩或最重要的东西。
